## Python to ascii

Έπειτα με ένα πολύ απλό `python` script θα μετατρέψω τους αριθμούς σε γράμματα ώστε να πάρουμε το flag. Παρακάτω ο python κώδικας. 

```python
to_ascii = [ 72, 84, 66, 123, 77, 48, 78, 107, 115, 95, 107, 78, 51, 119, 33, 125 ]

for i in to_ascii:
    print(chr(i), end =" ")
```
